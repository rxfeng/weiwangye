详情

税务e通网络发票（暂未上线）

  1. 基本信息

    产品类型

    政企业务

    简介

    “税务e 通-网络发票”是中国电信利用云计算平台为企业纳税人及税务管理机构提供网络发票及其它涉税业务的信息化应用。企业客户可通过远程在线的方式利用电脑、手机（蓝牙打印机）进行方便的发票开具、补打、作废、补录、冲红、验旧、查询、统计等工作，更方便更快捷；同时，利用有线及无线的虚拟安全专网技术充分保证了客户用户网络发票数据上传和下载的安全性。网络发票应用确保了消费者获取发票的真实性，有助于消费者维护自身权益，提升了税务部门为企业客户提供涉税服务的工作效率。

  2. 内容有效期

    开始日期

    2012-06-29 00:00:00

  3. 资费说明

    资费介绍

    • 1、功能费：根据终端不同分为PC使用功能费和手机使用功能费两种，目前功能费均免费。

    • 2、安全接入功能费：安全接入功能费指客户使用安全接入服务功能的费用，分为固网VPDN安全接入费、无线VPDN安全接入费两种，均为25元/VPDN账户/月。

    • 3、基本通信费：基本通信费指话音、短信、宽带接入、网络流量等费用。可根据客户通信消费需求、结合本地资费情况进行针对性设计。

  4. 客服流程

    客服流程

    资讯流程如用户咨询，业务代表可按以上内容给用户答复，“网络发票业务正在系统部署过程中，目前暂未上线，请您后期再与我们联系。”
